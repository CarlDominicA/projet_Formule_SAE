





            Ici se trouvent les codes pour la communication SPI. PI_MASTER_V3.py affiche dans un terminal
            les données envoyées par le ESP32-C3. PI_MASTER_V3_AFF.py fait la même chose que le premier,
            mais les données sont affichées avec la librairie Turtle. PI_MASTER_V3_AFF_BTN.py ne fait
            que rajouter la lecture et l'affichage de l'état du bouton.
